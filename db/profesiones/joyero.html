Joyero

Requerimientos para subir puntos en la profesión, y utilizar sus habilidades
  • ♦ 4 puntos en Agilidad
  • ♦ 3 puntos en Inteligencia
  • ♦ Conocimiento de Joyería
  • ♦ Conocimiento de Manualidades
  • ♦ Libro de joyería
  • ♦ Molde de joyas (Anillo, collar, o pendientes)
Para obtener puntos en esta profesión, debes:
  • ♦ Crear joyería.
  • ♦ Crear armazones para joyas.

Habilidades

Nivel 0. Puede confeccionar joyas de Nivel 1.
Nivel 2. Puede apilar gemas, joyas y fragmentos hasta dos (tres para fragmentos) por nivel en la profesión.
Nivel 4. Puede mejorar las joyas hasta Nivel 2.
Nivel 5. Pueden unir 4 fragmentos de gema, para crear la gema nuevamente.
Nivel 6. Las joyas creadas serán de Nivel 2.
Nivel 8. Puede platinar joyas, para subirlas a Nivel 3.
Nivel 12. Las joyas creadas serán de Nivel 3.
Nivel 15. Puede crear Joyas de Dones.
